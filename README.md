# Pong Game

A simple Pong game implemented in Python using Pygame.

## Dependencies

*   Pygame

To install Pygame, run:

```bash
pip install pygame
```

## How to Run

1.  Save the code as `pong.py`.
2.  Run the game from the command line:

```bash
python pong.py
```

## Controls

*   Player 1: W/S keys to move up and down.
*   Player 2: Up/Down arrow keys to move up and down.
